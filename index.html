HTML

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Use this site to find color hex codes.
  • Use MDN Docs to see what other properties and values to try.

Git

  • Use github flow.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • find out more about github copilot.
  • learn more about vscode debugging interface and debugging tricks in general.
  • any time you go to main branch run pull origin command.
  • The order in which one completes git flow. Always make sure you're in the correct branch with git status. Step 1. stage with git add -A 2. commit to repo with git commit -m "comment" 3. update current branch with git pull origin main 4. push up changes with git push origin branch-name.
